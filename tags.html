Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most impotant ones.

Document Tags

Tag Name Purpose
html HyperTextMarkupLanguage A standardized system for taggin text files
head head head is a wrapper for style sheets, meta tags and title tags
meta meta used to specify charset unicode
title title Used to label the tab of the webpage
body body Everything inside the body is displayed on the webpage

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (leveles 1–6)
p paragraph include a table
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
header Header of articles and sections Make a more structured page
div Acts as a container Adds more page structure
ol Ordered List Numbers list items
ul Unordered List Bulleted list items
li List Item Used hand and hand with ol or ul
blockquote Blockquote Used for longer quoations

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong <strong>at least a billion people</strong> atleast a billion people
a anchor make hyperlink <a href="https://learnenough.com/">Learn Enough</a> Learn Enough
img image include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
span span used as an inline container <span>Inline text here</span> Inline text here
